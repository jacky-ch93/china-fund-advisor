# 国内基金智能投顾助手

> 每次交易日 08:30 盘前推送 + 22:30 盘后总结

## 功能

- 📈 **盘前分析**：外围市场（美股/外汇/大宗）+ A股竞价数据 + 持仓参考
- 📉 **盘后总结**：今日收益 + 热点新闻 + 明日展望 + 自动更新飞书文档
- 🔧 **持仓管理**：纯文本录入，无需截图
- ⚡ **并行加速**：8只基金查询 0.3 秒完成

## 首次安装配置

安装后首次运行时会自动检测持仓——如果找不到持仓数据，会引导你：

1. **方式一**：提供已有的飞书持仓文档链接，助手自动提取数据并迁移
2. **方式二**：助手创建模板文档，你填入基金代码/金额后保存

## 安装

```bash
clawhub install china-fund-advisor
```

## 目录结构

```
china-fund-advisor/
├── SKILL.md              # 核心指令（触发词、命令格式）
├── metadata.json         # Skill 元数据
├── README.md             # 本文件
├── scripts/
│   ├── market.py         # 市场数据（大盘/外围/新闻）
│   ├── fund_data.py      # 基金净值（并行查询）
│   ├── portfolio.py     # 持仓增删改
│   ├── pre_market.py    # 盘前分析
│   ├── post_market.py   # 盘后总结+新闻
│   └── update_doc.py    # 飞书文档更新
└── templates/
    └── portfolio.md     # 持仓文档模板
```

## 触发命令

| 命令 | 说明 |
|------|------|
| `添加持仓：代码 金额 [名称]` | 录入/更新持仓 |
| `查看持仓` | 显示当前所有持仓 |
| `删除XXX` | 移除某只基金 |
| `清空持仓` | 删除所有持仓 |
| `跑一下盘前分析` | 立即生成盘前报告 |
| `跑一下盘后分析` | 立即生成盘后报告 |
| `诊断一下我的持仓` | 风险诊断+优化建议 |

## 数据来源

- 大盘/基金净值：AkShare（东方财富/天天基金）
- 热点新闻：东方财富快讯 API
- 汇率/大宗：新浪 API

## 安全声明

- 数据来自公开市场信息，不构成投资建议
- 基金估算收益存在误差，实际盈亏以支付宝/天天基金为准
- 持仓数据仅存储在本地，不上传任何第三方
