# 国内基金投顾助手

## 核心特点

- **无需截图**：直接接收基金代码+金额，纯文本操作
- **多源数据**：优先akshare → 新浪API → 缓存，多重保护
- **轻量快速**：并行查询，8只基金0.3秒全拿到
- **定时推送**：每个交易日08:30盘前 + 22:30盘后（含热点新闻+明日展望）
- **飞书文档同步**：盘后自动更新飞书持仓文档

## 飞书持仓文档

**文档地址**：`https://feishu.cn/docx/BltvdVn1moGnOAxT3Qtceg15nqg`
**用途**：每日22:30盘后自动更新，展示当前持仓明细与最新净值/涨跌幅

> 持仓数据以本地配置为准（portfolio.json），飞书文档为每日同步镜像。
> 如需修改持仓，请通过本助手命令操作（见下文），操作后会自动同步文档。

## 持仓管理

### 录入/更新持仓

```
帮我添加持仓：012922 47179.01 易方达全球成长精选混合(QDII)C
```

格式：`基金代码 金额`，多只用换行分隔。金额支持`万`/`元`，如`1.5万`=15000元。

**注：基金代码不足6位会自动补0。**

### 查询/修改持仓

| 命令 | 说明 |
|------|------|
| `添加持仓：代码 金额 [名称]` | 录入或更新持仓 |
| `删除XXX` | 移除某只基金 |
| `查看持仓` | 显示当前所有持仓 |
| `清空持仓` | 删除所有持仓 |
| `持有人设为XXX` | 设置持有人名称 |

> 每次持仓变更后，盘后推送时会自动同步更新飞书文档。

## 盘前报告内容

📈 **盘前分析** = 以下全部模块：

1. **🌏 外围市场** — 昨夜美股收盘（标普500/纳斯达克/道琼斯）
2. **💱 汇率** — 美元/离岸人民币实时汇率
3. **🛢️ 大宗商品** — 纽约黄金、白银、WTI原油
4. **🏛️ A股大盘** — 今日竞价/昨收涨跌（科创50/创业板/沪深300/上证/深证）
5. **💰 持仓基金参考** — 昨日净值及单日涨跌

## 盘后报告内容

📉 **盘后总结** = 以下全部模块：

1. **今日大盘** — 指数收盘表现 + 市场情绪
2. **📰 热点新闻** — 当日晚间宏观/股市/基金重要资讯
3. **行业表现** — 强势/弱势板块
4. **持仓基金表现** — 今日净值估算 + 各基操作建议
5. **📅 明日展望** — 今日总结 + 明日关注方向与风险提示
6. **飞书文档自动更新**

## 数据来源与稳定性

| 数据 | 优先 | 备选 | 保护 |
|------|------|------|------|
| 大盘指数 | akshare | 新浪API | 缓存 |
| 基金净值 | akshare（并行） | 同花顺 | 缓存 |
| 涨停数据 | akshare | - | 缓存 |
| 行业板块 | akshare | - | 缓存 |
| 热点新闻 | 东方财富快讯API | - | - |

基金查询采用**并行加速**，8只基金同时查询，总耗时约0.3秒。

任何单一接口失败不影响整体输出，无数据时显示"N/A"而非报错。

## 安全声明

- 所有数据来自公开市场信息，不构成投资建议
- 基金估算收益为根据持仓金额推算，存在一定误差
- 盘前数据存在盘中变化风险，请以实际净值为准
- 实际盈亏以支付宝/天天基金等平台为准
